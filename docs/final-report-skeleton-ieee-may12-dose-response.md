# Superseded: IEEE Final Report Skeleton

This file is no longer the active May 12 report scaffold.

Use [`docs/final-report-outline-acm-may12.md`](final-report-outline-acm-may12.md)
instead. The final report should follow ACM conference format, fit in 4-6 pages
with a maximum of 6 pages excluding references, and include the GitHub repository
link plus the YouTube demo link.

Do not copy wording from the old IEEE skeleton into the final report. It used the
wrong page budget and included historical process notes that should not appear in
the public paper.
